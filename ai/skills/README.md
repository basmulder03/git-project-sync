# Skills Library

This folder contains reusable implementation skills for coding agents.

Use these skills to speed up execution while preserving project safety rules.

## How to Use

1. Pick a skill from `ai/skills/skills.yaml`.
2. Read the skill file before implementing related tasks.
3. Apply the checklist in that skill to your current task.
4. Report completed checklist items in your handoff.

## Important

- Skills do not override `AGENTS.md` safety rules.
- Skills do not replace acceptance criteria in `docs/engineering/acceptance-test-matrix.md`.
- Skills must follow logical commit discipline from `ai/RUNBOOK.md`.
